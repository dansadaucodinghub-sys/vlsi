# sr_latch_behavioral

## Description

Level-sensitive SR latch with enable, asynchronous reset, set, reset, hold, and invalid state behavior.

## Modeling Style

Behavioral modeling.

## Files

| File | Description |
|---|---|
| `sr_latch_behavioral.v` | Verilog design |
| `sr_latch_behavioral_tb.v` | Verilog testbench |
| `Makefile` | VCS/Verdi simulation flow |

make all
