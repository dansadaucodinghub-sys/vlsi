# universal_shift_register_behavioral

## Description

4-bit universal shift register with hold, shift right, shift left, and parallel load operations.

## Modeling Style

Behavioral modeling.

## Files

| File | Description |
|---|---|
| `universal_shift_register_behavioral.v` | Verilog design |
| `universal_shift_register_behavioral_tb.v` | Verilog testbench |
| `Makefile` | VCS/Verdi simulation flow |

## Run

```bash
make all
```

## View Waveform

```bash
make waves
```
