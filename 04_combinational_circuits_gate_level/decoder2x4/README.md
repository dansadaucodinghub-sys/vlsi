# Decoder2X4 - Gate-Level Modeling

## Description

This design implements `decoder2x4` using Verilog gate-level modeling.

The design file uses primitive gates and avoids dataflow `assign` statements.

## Files

| File | Description |
|---|---|
| `decoder2x4.v` | Gate-level design |
| `decoder2x4_tb.v` | Testbench |
| `Makefile` | VCS / Verdi simulation flow |

## Run

```bash
make all
```

## Waveform

```bash
make waves
```
